🏷️实体类型体系

每个实体标注 type 和可选的 attributes：

  • 实体类型（7 大类）：person / object / animal / place / event / abstract / other
  • 属性类别（6 大类）：color / appearance / state / material / ordinal / other

规则：仅提取文本中明确出现的词汇（不做推断）；合并指代同一实体的重复项；排除代词和功能词。

🔗关系类型体系

每条关系为 subject → predicate → object 三元组，按 5 种类型分类：

  • geometric：空间定位（on, in, near, above, behind）
  • possessive：归属/部分-整体（has, of, with, wearing）
  • semantic：动作/交互（holding, playing, pushing）
  • temporal：时间关系（after, before, during, when）
  • other：不属于以上类型

交叉校验：关系同时通过 idx 和 name 引用实体，后处理检查索引范围和名称匹配，标记 valid 状态。
